AuthApi

All URIs are relative to http://localhost:3030/api

Method HTTP request Description
login POST /session Logins the user to the application, returning a new auth token and the user role.
logout DELETE /session Destroys the given auth token.

# login > SuccessfulLoginResponse login(loginRequest)

Logins the user to the application, returning a new auth token and the user role.

Example

// Import classes:
//import io.swagger.client.ApiException;
//import io.swagger.client.api.AuthApi;


AuthApi apiInstance = new AuthApi();
LoginRequest loginRequest = new LoginRequest(); // LoginRequest | 
try {
    SuccessfulLoginResponse result = apiInstance.login(loginRequest);
    System.out.println(result);
} catch (ApiException e) {
    System.err.println("Exception when calling AuthApi#login");
    e.printStackTrace();
}
